# URL Shortener Frontend

React + Vite frontend for URL shortener.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

App runs on `http://localhost:5173`

## Features

- Login / Register
- Create short links
- View link analytics
- Copy short URL to clipboard

## Stack

- React 19
- Vite
- React Router
- Axios
